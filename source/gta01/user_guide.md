

# MIC 麦克风板硬件用户指南

> **文档版本**：V1.0 &nbsp;&nbsp;|&nbsp;&nbsp; **适用产品系列**：ASN_4MIC 系列麦克风阵列板

## 1 概述

### 1.1 功能简介

ASN_4MIC 系列麦克风板是专为远场语音采集设计的麦克风阵列模组，每块板载 **4 颗底进音 MEMS 麦克风**，用于实现多通道拾音、声源定位及降噪等语音前处理功能。

麦克风板通过级联接口与主板相连，支持**单级**或**多级级联**部署，以满足不同阵列规模和拾音场景的需求。

## 2 板卡布局与模块说明

### 2.1 通用模块定义

所有 MIC 板均包含以下三类核心模块：

| 模块 | 标识 | 功能说明 |
|:----:|:---:|:--------|
| **接口 A** | JP（橙色标注） | **上行接口** —— 连接主板或上一级 MIC 板。当仅使用单级 MIC 板时，只需连接此接口。 |
| **接口 B** | JP（蓝色标注） | **下行接口** —— 连接下一级 MIC 板（级联扩展时使用）。 |
| **麦克风 × 4** | MIC（红色标注） | 4 颗底进音 MEMS 麦克风，均匀分布于板面。 |

> **"+"极性标识**：每个接口旁有一个**圆点标记**，圆点所在一侧即为接口的 **"+"（正极）方向**。接线时务必对准正极方向，防止反接。

### 2.2 各型号板卡布局详情

#### 2.2.1 环形 MIC 板 — BOD0038（ASN_4MIC_H6CM_70_70）

- **板型特征**：圆形 PCB，四角带切平安装边，4 个 M2 安装孔。
- **接口 A（上行）**：标注为 **JP2**，位于板面右上方；**左侧圆点**方向为"+"。
- **接口 B（下行）**：标注为 **JP1**，位于板面左上方；**右侧圆点**方向为"+"。
- **麦克风分布**：4 颗 MIC 呈圆形阵列等间距排列，间距约 6 cm。

#### 2.2.2 线形 MIC 板 — BOD0043（ASN_4MIC_4CM_154_15）

- **板型特征**：长条形 PCB，尺寸 154 × 15 mm，2 个 M2 安装孔。
- **接口 A（上行）**：**下方圆点**方向为"+"。
- **接口 B（下行）**：**上方圆点**方向为"+"。
- **麦克风分布**：4 颗 MIC 沿长边线性排列，间距 4 cm。

#### 2.2.3 线形 MIC 板 — BOD0044（ASN_4MIC_3.5CM_160_17）

- **板型特征**：长条形 PCB，尺寸 160 × 17 mm，2 个 M2 安装孔。
- **接口 A（上行）**：**右侧圆点**方向为"+"。
- **接口 B（下行）**：**右侧圆点**方向为"+"。
- **麦克风分布**：4 颗 MIC 沿长边线性排列，间距 3.5 cm。

#### 2.2.4 线形 MIC 板 — BOD0046（ASN_4MIC_3.5CM_115_20）

- **板型特征**：长条形 PCB，尺寸 115 × 20 mm，2 个 M2 安装孔。
- **接口 A（上行）**：**上方圆点**方向为"+"。
- **接口 B（下行）**：**下方圆点**方向为"+"。
- **麦克风分布**：4 颗 MIC 沿长边线性排列，间距 3.5 cm。

## 3 硬件连接

### 3.1 单级连接（1 块 MIC 板）

最简配置，适用于基础 4 通道拾音场景。

```
┌──────────┐       排线        ┌────────────┐
│   主板   │ ──────────────▶  │  MIC 板    │
│  (HOST)  │   "+"对"+"     │  接口 A    │
└──────────┘                  └────────────┘
```

**操作步骤**：

1. 取排线，确认两端方向。
2. 将排线一端接入**主板**的 MIC 输出接口。
3. 将排线另一端接入 MIC 板的**接口 A**，确保"+"极性标记一致。
4. 接口 B 悬空，无需连接。

### 3.2 多级级联连接（≥2 块 MIC 板）

当需要扩展至 8 路或更多通道的麦克风阵列时，使用级联方式。

```
┌──────────┐      排线      ┌──────────────┐      排线      ┌──────────────┐
│   主板   │ ────────────▶ │ MIC 板 #1    │ ────────────▶ │ MIC 板 #2    │
│  (HOST)  │  "+"对"+"   │ 接口A  接口B │  "+"对"+"   │ 接口A  接口B │──▶ ...
└──────────┘               └──────────────┘               └──────────────┘
```

**操作步骤**：

1. 将主板通过排线连接至**第 1 级 MIC 板的接口 A**（"+"对"+"）。
2. 将第 1 级 MIC 板的**接口 B** 通过排线连接至**第 2 级 MIC 板的接口 A**（"+"对"+"）。
3. 如需继续扩展，重复上述级联步骤。
4. **最末级** MIC 板的接口 B 悬空。

### 3.3 连接注意事项

| 项目 | 说明 |
|:----:|:-----|
| ⚠️ 极性 | 接线时必须对齐"+"圆点标记，反接可能导致通信异常或器件损坏。 |
| ⚠️ 上电顺序 | 建议先完成所有排线连接，再给主板上电。 |
| ⚠️ 安装 | 使用 M2 螺丝通过安装孔固定 MIC 板，确保麦克风进音孔（底部）不被遮挡。 |
| ⚠️ 进音孔 | 麦克风为**底进音**设计，安装时需确保 PCB 底面进音孔与外壳/结构件的拾音孔对准。 |

## 4 典型应用场景

### 4.1 环形阵列（BOD0038）

- **典型产品**：智能音箱、桌面会议终端、语音机器人。
- **特点**：麦克风呈圆形分布，适合 **360° 全向拾音** 和声源定位。
- **建议部署**：水平放置于产品顶部，拾音孔朝下对准外壳开孔。

### 4.2 线形阵列（BOD0043 / BOD0044 / BOD0046）

- **典型产品**：智能电视 / Soundbar、智能面板、壁挂式语音终端。
- **特点**：麦克风沿直线排列，适合 **定向波束成形**，有效抑制两侧噪声。
- **建议部署**：沿产品长边水平安装，拾音方向正对用户。

### 4.3 型号选择建议

| 应用需求 | 推荐型号 | 理由 |
|:---------|:--------:|:-----|
| 360° 全向拾音 | BOD0038 | 环形阵列，声源定位覆盖全方位 |
| 长条形产品、空间充裕 | BOD0043 | 4cm 间距，频率响应更优 |
| 长条形产品、窄边框 | BOD0044 | 3.5cm 间距，板宽仅 17mm |
| 紧凑型产品 | BOD0046 | 板长仅 115mm，适配小尺寸产品 |

## 附录

### A. 术语说明

| 术语 | 含义 |
|:----:|:-----|
| MIC | 麦克风（Microphone） |
| MEMS | 微机电系统，此处指 MEMS 麦克风传感器 |
| 底进音 | 麦克风拾音孔位于 PCB 底面 |
| 级联 | 多块 MIC 板串联连接以扩展通道数 |
| 排线 | 用于板间连接的柔性扁平电缆（FFC/FPC） |
